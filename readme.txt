This is a demo code for the quality assessment of the polarization images proposed in our following TIP paper. If you use this code or it is helpful for your research, please cite our work:
N. Li, B. Le Teurnier, M. Boffety, F. Goudail, Y. Zhao and Q. Pan, "No-Reference Physics-Based Quality Assessment of Polarization Images and its Application to Demosaicking," in IEEE Transactions on Image Processing, doi: 10.1109/TIP.2021.3122085.

====================================================================================================

-Code:
	- demo.m : Run this code to see how to call the function that calculate the APMR criterion and predict the quality of the example input polarizaiton images by 		adding Gaussian noise.
	- APMR_compute.m : Function for computing the APMR criterion.
	- example.mat : An example of ideal input polarization images for testing the code.
